Условия JS

Условные операторы используются для выполнения различных действий в зависимости от условий.

Условные заявления

В JS есть следующие условные выражения: 
		if       - для указания блока кода, который должен быть выполнен, если указанное условие истинно 
		else     - для указания блока кода, который будет выполнен, если то же условие ложно 
		else if  - чтобы указать новое условие для проверки, если первое условие ложно 
		switch   - для указания множества альтернативных блоков кода, которые должны быть выполнены

Заявление if

if оператор - используется, чтобы указать блок кода JS, который будет выполнен, если условие истинно.
СИНТАКСИС 
		if (condition) {
		   //  block of code to be executed if the condition is true
		}

ПРИМЕР 
		if (hour > 19) {
		   greeting = "Good day";
		}
		//
		if (new Date().getHours() > 19) {
			document.getElementById("if_0").innerHTML = "Good day";
		}

Заявление else

else оператор - используется, чтобы указать блок кода, который будет выполнен, если условие ложно.
СИНТАКСИС 
		if (condition) {
		   //  block of code to be executed if the condition is true
		} else {
		   //  block of code to be executed if the condition is false
		}

ПРИМЕР 
		var hour = new Date().getHours();
		var greeting;
		if (hour < 12) {
			greeting = "Good morning";
		} else {
			greeting = "Good day";
		}
		document.getElementById("if_else_0").innerHTML = greeting;

Заявление else if

else if оператор - используется, чтобы указать новое условие, если первое условие ложно.
СИНТАКСИС 
		if (condition1) {
		   //  block of code to be executed if condition1 is true
		} else if (condition2) {
		   //  block of code to be executed if the condition1 is false and condition2 is true
		} else {
		   //  block of code to be executed if the condition1 is false and condition2 is false
		}

ПРИМЕР 
		var hour = new Date().getHours();
		var greeting;
		if (hour < 12) {
		   greeting = "Good morning";
		} else if (hour < 16){
		   greeting = "Good day";
		} else {
		   greeting = "Good evening"
		}
		document.getElementById("if_else_0").innerHTML = greeting;

Заявление switch

Оператор switch используется для выполнения различных действий в зависимости от условий.

Заявление о переключении JS

switch инструкция, используется чтобы выбрать один из множества блоков кода, которые будут выполнены.
СИНТАКСИС 
		switch(expression) {
		  case x:
		    // code block
		    break;
		  case y:
		    // code block
		    break;
		  default:
		    // code block
		}
Как работает. Выражение переключателя вычисляется один раз. Значение выражения сравнивается со значениями каждого случая.
Если есть совпадение, выполняется соответствующий блок кода. Если совпадений нет, выполняется блок кода по умолчанию.
ПРИМЕР 
		var day;
		switch (new Date().getDay()) {
		   case 0:
		     day = "Sunday";
		     break;
		   case 1:
		     day = "Monday";
		     break;
		   case 2:
		     day = "Tuesday";
		     break;
		   case 3:
		     day = "Wadnesday";
		     break;
		   case 4:
		     day = "Thursday";
		     break;
		   case 5:
		     day = "Friday";
		     break;
		   case 6:
		     day = "Saturday";
		     break;
		}

Ключевое слово break

Когда JS достигает ключевого слова break, оно выходит из блока переключателей. Это остановит выполнение внутри блока.
Нет необходимости разбивать последний случай в блоке переключателей. Блок все равно ломается.
Если пропустить оператор break, будет выполнен следующий случай, даже если оценка не соответствует случаю.

Ключевое слово default

default - Ключевое слово указывает код для запуска, если нет ни одного случая матча.

ПРИМЕР 
		getDay() Метод возвращает день недели как число между 0 и 6.
		Если сегодня не суббота (6) и не воскресенье (0), можно написать сообщение по умолчанию:
		var text;
		switch (new Date().getDay()) {
		   case 6:
		     text = "Today is Saturday";
		     break;
		   case 0:
		     text = "Today is Sunday";
		     break;
		   default:
		     text = "Looking forward to the Weekend";
		}
		

		Если default это не последний случай в блоке переключателей, нужно завершить случай по умолчанию с break.
		var text;
		switch (new Date().getDay()) {		
		   default:
		     text = "Looking forward to the Weekend";
		     break;
		   case 6:
		     text = "Today is Saturday";
		     break;
		   case 0:
		     text = "Today is Sunday";
		     break;
		}

Общие кодовые блоки

Если нужно, чтобы разные варианты переключения использовали один и тот же код.
ПРИМЕР 
		var text;
		switch (new Date().getDay()) {			
		  case 4:
		  case 5:
		    text = "Soon it is Weekend";
		    break;
		  case 0:
		  case 6:
		    text = "It is Weekend";
		    break;
		  default:
		    text = "Looking forward to the Weekend";
		}

Детали переключения

Если несколько cases соответствуют значению case, выбирается первый случай.
Если подходящих вариантов не найдено, программа переходит к метке default.
Если метка default не найдена, программа переходит к инструкциям после переключения.

Строгое сравнение

Переключать case(случаи) используют строгое сравнение (===). Значения должны быть одного типа для соответствия.
Строгое сравнение может быть верным, только если операнды имеют одинаковый тип.
ПРИМЕР 
		В этом примере не будет совпадения для x:
		var x = "0";
		switch (x) {
		  case 0:
		    text = "Off";
		    break;
		  case 1: 
		    text = "On";
		    break;
		  default:
		    text = "No value found";
		}
