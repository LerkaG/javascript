Битовые операторы JavaScript


ПРИМЕР 
		&   (И)   - возвращает 1, если оба бита равны 1.               
		|   (ИЛИ) - возвращается 1, если один из битов равен 1.       
		^   (XOR) - возвращается 1, если биты разные.  
		~   (НЕ)  - инвертирует все биты.
		<<  (Нулевое заполнение левого смещения) - Сдвиг влево, толкая нули справа и оставляя самые левые биты. 
		>>  (Подпись вправо сдвиг) - Сдвиг вправо, толкая копии самого левого бита слева, и пусть самые правые биты падают
		>>> (Нулевое заполнение вправо сдвиг) - Сдвиг вправо, толкая нули слева, и пусть самые правые биты падают
		
		5 & 1;   //
		5 | 1;   //
		5 ^ 1;   //
		  ~ 5;   //
		5 << 1;  //
		5 >> 1;  //
		5 >>> 1; //

JS (нулевое заполнение) битовое смещение влево <<

Это смещение нуля влево. Один или несколько нулевых битов вставляются справа, и самые левые биты падают:
ПРИМЕР 
		15 << 10;  //

JS (сохранение знака) Битовый сдвиг вправо >>

Это знак сохранения правого сдвига. Копии самого левого бита вставляются слева, а самые правые биты падают:
ПРИМЕР 
		-5 >> 1; //

JS (нулевая заливка) вправо Shift >>>

Это смещение нуля вправо. Один или несколько нулевых битов вставляются слева, а самые правые биты падают:
ПРИМЕР 
		5 >>> 1; //

Преобразование десятичного числа в двоичное


ПРИМЕР 
		function dec_to_bin(dec) {
		   return (dec >>> 0).toString(2);
		}
		dec_to_bin(40); //

Регулярные выражения JavaScript

Регулярное выражение - это последовательность символов, образующая шаблон поиска .

Что такое регулярное выражение?


СИНТАКСИС 
		/pattern/modifiers;

ПРИМЕР 
		var patt = /w3schools/i;
/ w3schools / i - это регулярное выражение. w3schools - это шаблон (для поиска).
i - является модификатором (модифицирует поиск, чтобы быть нечувствительным к регистру).

Использование строковых методов

В JS регулярные выражения часто используются с двумя строковыми методами : search() и replace().
search() - Метод использует выражение для поиска соответствия, и возвращает позицию матча.
replace() - Метод возвращает модифицированную строку , в которой заменяется шаблон.

Использование строки search() со строкой

В search() методе ищет строку для указанного значения и возвращает позицию матча:
ПРИМЕР 
		var str = "Visit schools!";
		var n = str.search("schools");
		n; //

Использование строки search() с регулярным выражением


ПРИМЕР 
		var str = "Visit schools!";
		var n = str.search(/schools/i);
		n; //

Использование строки replace()

replace() - Метод заменяет указанное значение с другим значением в строке
ПРИМЕР 
		var str_1 = "Visit Microsoft!";
		var res = str_1.replace("Microsoft", "schools");
		res;
		 
		Visit Microsoft!

Использование строки replace() с регулярным выражением

Используйте регулярное выражение без учета регистра, чтобы заменить Microsoft на W3Schools в строке:
ПРИМЕР 
		function func_replace_1() {
		   var str = document.getElementById("reg_3").innerHTML;
		   var res = str.replace(/microsoft/i, "Schools");
		   res;
		}
		 
		Please visit Microsoft and Microsoft!

Модификаторы регулярных выражений

Модификаторы можно использовать для выполнения глобальных поисков без учета регистра:
i - Выполнить сопоставление без учета регистра
g - Выполнить глобальное совпадение (найти все совпадения, а не останавливаться после первого совпадения)
m - Выполнить многострочное сопоставление
ПРИМЕР 
		function func_match_0() {
		   var str = document.getElementById("match_0").innerHTML;
		   var txt = /microsoft/i;
		   var res = str.match(txt);
		   res;
		}
		 
		Please visit Microsoft and Microsoft!
		
		function func_match_1() {
		   var str = "Please visit Microsoft and Microsoft!"
		   var txt = /microsoft/g;
		   var res = str.match(txt);
		   res;
		}
		 
		Please visit Microsoft and Microsoft!

		function func_match_2() {
		   var str = "Please visit Microsoft and Microsoft!"
		   var txt = /^microsoft/m;
		   var res = str.match(txt);
		   res;
		}
		 
		Please visit Microsoft and Microsoft!

Шаблоны регулярных выражений

Скобки используются для поиска диапазона символов: [abc], [0-9], (x|y)
Метасимволы - это символы с особым значением: \d, \s, \b, \uxxxx
ПРИМЕР 
		[abc] - Найдите любой из символов в скобках 
		var str = "Please visit Microsoft and Microsoft!"
		var txt = /[e]/g;
		var res = str.match(txt);
		res; 
		Please visit Microsoft and Microsoft!

		[0-9] - Найдите любую из цифр в скобках
		var str = "1234567890"
		var txt = /[1-5]/g;
		var res = str.match(txt);
		res; 
		1234567890

		(x|y) - Найдите любую из альтернатив, разделенных | 
		var str = "re, green, red, green, gren, gr, blue, yellow"
		var txt = /(red|green)/g;
		var res = str.match(txt);
		res; 
		re, green, red, green, gren, gr, blue, yellow

		Метасимволы - это символы с особым значением:
		\d - Найти цифру
		var str = "Give 100%"
		var txt = /\d/g;
		var res = str.match(txt);
		res;
		
		Give 100%

		\s - Найти пробел
		var str = "Is this all there is?"
		var txt = /\s/g;
		var res = str.match(txt);
		res;
		 Is this all there is?

		\b - Найти совпадение в начале первого слова - \bWORD, в конце первого слова -  WORD\b.
		var str = "HELLO, LOOK AT YOU!";

		var txt_start = /\bLO/;
		var res_start = str.search(txt_start);
		"start: " + res_start;
		HELLO, LOOK AT YOU!
		
		var txt_end = /LO\b/;
		var res_end = str.search(txt_end);
		"end: " + res_end;
		HELLO, LOOK AT YOU!
		

		\uxxxx - Найти символ Unicode, указанный шестнадцатеричным числом xxxx
		var str = "Visit W3Schools. Hello World!";
		var patt = /\u0057/g;
		var res = str.match(patt);
		res; 
		Visit W3Schools. Hello World!

		Квантификаторы определяют количества: n+, n*, n? 
		n+ - Соответствует любой строке, содержащей хотя бы одно n
		var str = "Visit W3Schools. Hello World!";
		var patt = /o+/g;
		var res = str.match(patt);
		res; 
		Visit W3Schools. Hello World!

		n* - Соответствует любой строке, которая содержит ноль или более вхождений n
		var str = "1, 100 or 1000?";
		var patt = /10*/g;
		var res = str.match(patt);
		res; 
		1, 100 or 1000?

		n? - Соответствует любой строке, которая содержит ноль или одно вхождение n
		var str = "1, 100 or 1000?";
		var patt = /10?/g;
		var res = str.match(patt);
		res; 
		1, 100 or 1000?

Использование объекта RegExp

В JS объект RegExp является объектом регулярного выражения с предопределенными свойствами и методами.

Использование test()

test() - Способ представляет собой способ выражения RegExp. Он ищет строку для шаблона и возвращает true или false, в зависимости от результата.
ПРИМЕР 
		Поиск строки для символа «e»:
		Вариант 1:
		var patt = /e/;
		var result = patt.test("The best things");
		result;	
		The best things
		
		Вариант 2:
		var patt = /e/.test("The best things");
		patt;
		The best things

		Вариант 3:
		var text = document.getElementById("text").innerHTML;
		document.getElementById("test_1").innerHTML = /e/.test(text);
		< i id="text_0">"The best things"< /i>
		"The best things" // 

		Вариант 4:
		var text = /e/.test("The best things");
		text;
		"The best things" //

Использование exec()

exec() - Способ представляет собой способ выражения RegExp. Он ищет строку для указанного шаблона и возвращает найденный текст как объект.
Если совпадений не найдено, возвращается пустой (null) объект.

		var text_1 = /e/.exec("The best things");
		text_1; //

		var text_2 = /w/.exec("The best things");
		text_2; //
