Преобразование типов JS

Number() конвертирует в Number, String() конвертирует в String, Boolean() конвертирует в Boolean.

Типы данных JS

Есть 5 различных типов данных, которые могут содержать значения: string, number, boolean, object, function
Есть 6 типов объектов: Object, Date, Array, String, Number, Boolean
И 2 типа данных, которые не могут содержать значения: null и undefined

Тип оператора

Можно использовать typeof оператор, чтобы найти тип данных переменной JS.
Нельзя использовать, typeof чтобы определить, является ли объект JS массивом (или датой).

Тип данных typeof

typeof Оператор не является переменной. Это оператор. Операторы (+ - * /) не имеют данных любого типа.
Но typeof оператор всегда возвращает строку (содержащую тип операнда).

Свойство constructor

constructor Свойство возвращает функцию конструктора для всех переменных JS.
ПРИМЕР 
		Можно проверить свойство конструктора, чтобы узнать, является ли объект Array (содержит слово «Массив»)
		var fruits = ["Banana", "Orange", "Apple", "Mango"];
		function is_array(my_array) {
		   return my_array.constructor.toString().indexOf("Array") > -1;
		}
		is_array(fruits); // 

		Или еще, можно проверить, является ли объект функцией Array:
		function is_array_1(my_array) {
		   return my_array.constructor === Array;
		}
		is_array_1(fruits); // 

		Можно проверить свойство конструктора, чтобы узнать, является ли объект Date(содержит слово «Дата»):
		var my_date = new Date();
		function is_date(my_date) {
		   return my_date.constructor.toString.indexOf("Date") > -1;
		}
		is_date(my_date); // 

		function is_date_1(my_date) {
		   return my_date.constructor === Date;
		}
		is_date_1(my_date); //


    Преобразование типов JS

    Переменные JS могут быть преобразованы в новую переменную и другой тип данных: С помощью функции и Автоматически с помощью самого JS

    Преобразование чисел в строки

    Глобальный метод String() может преобразовывать числа в строки. Он может быть использован для любого типа чисел, литералов, переменных или выражений:
    ПРИМЕР 
    		var x = 123;
		String(x)        // 
		String(123)      // 
		String(100 + 23) // 

		Метод .toString() делает то же самое.
		x.toString();          // 
		(123).toString();      // 
		(100 + 23).toString(); //

    Преобразование логических значений в строки

    ПРИМЕР 
    		Глобальный метод String() может преобразовывать логические значения в строки.
		String(true)  // 
		String(false) // 

		Булевый метод String() делает то же самое.
		true.toString()  // 
		false.toString() //

    Преобразование дат в строки

    ПРИМЕР 
    		Глобальный метод String() может конвертировать даты в строки.
		String(Date());    // 

		Метод Date toString() делает то же самое.
		Date().toString(); //

    Преобразование строк в числа

    Глобальный метод Number() может преобразовывать строки в числа. Строки, содержащие числа (напр, «3.14»), преобразуются в числа (напр, 3.14).
    Пустые строки конвертируются в 0. Все остальное конвертируется в NaN(не число).
    ПРИМЕР 
    		Number("3.14");  // 
		Number(" ");     // 
		Number("");      // 
		Number("99 99"); //

    Унарный + Оператор

    Унарный оператор + может быть использован для преобразования переменного в число
    ПРИМЕР 
    		var y = "5";
		var x = + y;
		y; //  
		x; //  

		Если переменная не может быть преобразована, она все равно станет числом, но со значением NaN(не числом):
		var z = "John";
		var u = + z;
		u; //  
		z; //

    Преобразование логических чисел в числа

    Глобальный метод Number() также может преобразовывать логические значения в числа.
    ПРИМЕР 
    		Number(true);  //  
		Number(false); //

    Преобразование дат в числа

    Глобальный метод Number() может быть использован для преобразования дат в числа.
    ПРИМЕР 
    		var d = new Date();
		Number(d); // 

		Метод даты getTime() делает то же самое.
		d.getTime(); //

    Автоматическое преобразование типов

    Когда JS пытается работать с «неправильным» типом данных, он пытается преобразовать значение в «правильный» тип.
    ПРИМЕР 
    		Результат не всегда такой, который можно ожидать:
		5 + null;   // 
		"5" + null; // 
		"5" + 2;    // 
		"5" - 2;    // 
		"5" * "2";  // 
		"5" / "2";  //

    Автоматическое преобразование строк

    JS автоматически вызывает toString() функцию переменной, когда вы пытаетесь «вывести» объект или переменную:
    ПРИМЕР 
    		var my_var;
		my_var_0 = {name:"Fjohn"}; // 
		my_var_1 = [1,2,3,4];      // 
		my_var_2 = new Date();     // 

		Числа и логические значения также преобразуются, но это не очень заметно:
		my_var_3 = 123;   // 
		my_var_4 = true;  // 
		my_var_5 = false; //
