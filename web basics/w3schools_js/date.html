JavaScript Date Objects

По умолчанию JS будет использовать часовой пояс браузера и отображать дату в виде полнотекстовой строки
Вс 05 июл 2020 13:29:17 GMT + 0300 (Восточная Европа, летнее время)

Создание объектов Date

Объекты Date создаются с помощью new Date() конструктора. Есть 4 способа создать новый объект даты:
ПРИМЕР 
		new Date();
		new Date(year, month, day, hours, minutes, seconds, milliseconds);
		new Date(milliseconds);
		new Date(date string);

new Date()

new Date() - создает новый объект даты с текущей датой и временем.
Объекты Date являются статическими. Время компьютера тикает, а объекты даты - нет
ПРИМЕР 
		var d = new Date(); //

new Date(year, month, ...)

new Date(year, month, ...) - создает новый объект даты с указанной датой и временем. 7 чисел указывают год, месяц, день, час, минуту, секунду и миллисекунду (в указанном порядке).
JS считает месяцы от 0 до 11. Нельзя пропустить месяц. Если указать только один параметр, он будет считаться миллисекундами
ПРИМЕР 
		var new_date_1 = new Date(2020, 11, 24, 10, 33, 30, 0);
		new_date_1; //

		var new_date_2 = new Date(2020, 11, 24, 10, 33, 30);
		new_date_2; //

		var new_date_3 = new Date(2020, 11, 24, 10, 33);
		new_date_3; //

		var new_date_4 = new Date(2020, 11, 24, 10);
		new_date_4; //

		var new_date_5 = new Date(2020, 11, 24);
		new_date_5; //

		var new_date_6 = new Date(2020, 11);
		new_date_6; //

		var new_date_7 = new Date(2020);
		new_date_7; //

Предыдущий век

Год с одной и двумя цифрами будет интерпретироваться как 19xx
ПРИМЕР 
		var new_date = new Date(99, 11, 24);
		new_date; //

		var new_date = new Date(9, 11, 24);
		new_date; //

new Date(dateString)

new Date(dateString) - создает новый объект даты из строки даты
ПРИМЕР 
		var new_date = new Date("October 13, 2014 11:13:00");
		new_date; //

JavaScript хранит даты в миллисекундах

JS хранит даты в виде количества миллисекунд с 01 января 1970 года, 00:00:00 UTC (Всемирное координированное время).
Нулевое время 01 января 1970 г. 00:00:00 UTC.

new Date(milliseconds)

new Date(milliseconds) - создает новый объект даты как нулевое время плюс миллисекунды
(24 часа) составляет 86 400 000 миллисекунд
ПРИМЕР 
		var d = new Date(0);
		d; // 

		01 января 1970 года плюс 100 000 000 000 миллисекунд - это приблизительно 3 марта 1973 года:
		var d = new Date(100000000000);
		d; // 

		01 января 1970 года минус 100 000 000 000 миллисекунд составляет примерно 31 октября 1966 года:
		var d = new Date(-100000000000);
		d; // 

		var d = new Date(1595000000000);
		d; //

Отображение дат

JS по умолчанию выводит даты в формате полнотекстовой строки. При отображении объекта даты в HTML, он автоматически преобразуется в строку с помощью toString() метода.
toUTCString() - Метод преобразует дату в строку UTC (стандартный дисплей даты).
toDateString() - Метод преобразует дату в более читаемый формат
ПРИМЕР 
		var d = new Date();		

		Одинаковый результат
		d;            // 
		d.toString(); // 

		d.toUTCString();  // 
		d.toDateString(); //

Форматы дат JavaScript

JavaScript ввод/вывод даты

есть 3 типа форматов ввода даты в JavaScript:
Формат ISO соответствует строгим стандартам JS. Другие форматы не так хорошо определены и могут зависеть от браузера.
ПРИМЕР 
		Дата ISO    «2015-03-25» (Международный стандарт)
		Короткая дата "03/25/2015".
		Длинная дата  "Mar 25 2015" или "25 Mar 2015"

Вывод даты в JavaScript

Независимо от формата ввода, JS будет (по умолчанию) выводить даты в формате полнотекстовой строки
ПРИМЕР 
		Wed Mar 25 2015 02:00:00 GMT+0200 (Восточная Европа, стандартное время)

JS ISO 8601 даты

(ГГГГ-ММ-ДД) - Вычисленная дата будет относительно вашего часового пояса.
(ГГГГ-ММ) - Часовые пояса будут отличаться результатом выше между 28 февраля и 01 марта.
(ГГГГ) - Часовые пояса будут отличаться от указанных выше результатов в период с 31 декабря 2014 года по 01 января 2015 года.
ПРИМЕР 
		var d = new Date("2020-06-22");
		d; // 

		var d = new Date("2020-06");
		d; // 

		var d = new Date("2020");
		d; //

Даты ISO (Дата-Время)

Т - разделитель Даты и время. Z - определяется Время UTC.
UTC (всемирное координированное время) совпадает с GMT (среднее время по Гринвичу).
Пропуск T или Z в строке даты и времени может давать разные результаты в разных браузерах.
ПРИМЕР 
		var d = new Date("2015-03-25T12:00:00Z");
		d; // 

		var d = new Date("2015-03-25T12:00:00-06:30");
		d; //

Короткие Даты

Краткие даты записываются с синтаксисом «MM/DD/YYYY»
ПРИМЕР 
		var d = new Date("03/25/2015");
		d; // 

		В некоторых браузерах месяцы или дни без начальных нулей могут привести к ошибке
		var d = new Date("2015-3-25");
		d; // 

		Некоторые браузеры пытаются угадать формат. Некоторые вернут NaN.
		var d = new Date("2015/03/25");
		d; // 
		var d = new Date("25-03-2015");
		d; //

Длинные даты

Длинные даты чаще всего пишутся с синтаксисом «MMM DD YYYY». Месяц и день могут быть в любом порядке.
Месяц может быть записан полностью или сокращенно. Запятые игнорируются. Имена нечувствительны к регистру
ПРИМЕР 
		var d = new Date("25 Mar 2015");
		d; // 

		var d = new Date("Mar 25 2015");
		d; // 
		var d = new Date("March 25 2015");
		d; // 

		var d = new Date("MARCH, 25, 2015");
		d; //

Ввод даты - даты разбора

Date.parse() метод можно использовать если есть допустимая строка даты, для ее преобразования в миллисекунды.
ПРИМЕР 
		var msec = Date.parse("March 21, 2012");
		msec; // 

		Затем можно использовать количество миллисекунд, чтобы преобразовать его в объект даты
		var d = new Date(msec);
		d; // 

		var milsec = 345353453;
		var date_on_milsec = new Date(milsec);	
		date_on_milsec; //

Get Date Методы

Методы для получения информации от объекта даты
ПРИМЕР 
		getFullYear()     - Получить год в виде четырехзначного числа (гггг) 
		getMonth()        - Получить месяц (0-11) 
		getDate()         - Получить месяц (1-31) 
		getHours()        - Получить часы (0-23) 
		getMinutes()      - Получить минуты (0-59) 
		getSeconds()      - Получить секунды (0-59) 
		getMilliseconds() - Получить милисекунды (0-999) 
		getTime()         - Получить время (миллисекунды с January 1, 1970 ) 
		getDay()          - Получить день недели (0-6) 
		Date.now()        - Получить время. ECMAScript 5.

Методы

В JS первый день недели (0) означает «воскресенье», даже если некоторые страны мира считают первый день недели «понедельником».
ПРИМЕР 
		var date = new Date();
		var days = ["Sunday","Monday","Tuesday","Wednesday","Thursday","Friday","Saturday"];
		var month = ["January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"];

		date.getTime();         //
		date.getFullYear();     //

		date.getMonth();        //
		month[date.getMonth()]; //

		date.getDate();         //
		date.getHours();        //
		date.getMinutes();      //
		date.getSeconds();      //
		date.getMilliseconds(); //

		date.getDay();          //
		days[date.getDay()];    //

Set Date Методы

Методы Set Date позволяют устанавливать значения даты (годы, месяцы, дни, часы, минуты, секунды, миллисекунды) для объекта Date.

Методы установки даты

Методы Set Date используются для установки части даты
ПРИМЕР 
		setDate()         - Установите день как число (1-31)
		setFullYear()     - Установите год (необязательно месяц и день)
		setHours()        - Установите часы (0-23)
		setMilliseconds() - Установите милисекунды (0-999)
		setMinutes()      - Установите минуты (0-59)
		setMonth()        - Установите месяца (0-11)
		setSeconds()      - Установите секунды (0-59)
		setTime()         - Установите время (миллисекунды с January 1, 1970 )

Методы

setFullYear() - Метод может при необходимости установить месяц и день setDate() - метод также можно использовать для добавления дней к дате
ПРИМЕР 
		date.setDate(20);                  // 
		Если добавление дней сдвигает месяц или год, изменения обрабатываются объектом Date автоматически.
		date.setDate(date.getDate() + 20); // 

		date.setFullYear(2010);            // 
		date.setFullYear(2020, 11, 3);     // 

		date.setHours(4);                  // 
		date.setMinutes(50);               // 
		date.setMilliseconds(230);         // 
		date.setSeconds(20);               // 
		date.setTime(20);                  // 
		date.setMonth(10);                 //

Сравнить даты


ПРИМЕР 
		var today, someday, text;
		today = new Date();
		someday = new Date();
		someday.setFullYear(2100, 0, 14);
		if (someday > today) {
		   text = "Today is before January 14, 2100";
		} else {
		   text = "Today is after January 14, 2100";
		}
		text; //
