Math Object

Объект JS Math позволяет выполнять математические задачи над числами. 
		Math.PI; //

Math.round()

Math.round(x) - возвращает значение x, округленное до ближайшего целого числа
ПРИМЕР 
		Math.round(2.33); // 
		Math.round(2.63); //

Math.pow()

Math.pow(x, y) - возвращает значение x в степень y
ПРИМЕР 
		Math.pow(2, 4); //

Math.sqrt()

Math.sqrt(x) - возвращает квадратный корень из x
ПРИМЕР 
		Math.sqrt(4); //

Math.abs()

Math.abs(x) - возвращает абсолютное (положительное) значение x:
ПРИМЕР 
		Math.abs(-4.7); //

Math.ceil()

Math.ceil(x) - возвращает значение x, округленное до ближайшего целого числа
ПРИМЕР 
		Math.ceil(-4.7); // 
		Math.ceil(4.7);  //

Math.floor()

Math.floor(x) - возвращает значение х округляется вниз до ближайшего целого числа
ПРИМЕР 
		Math.floor(-4.7); // 
		Math.floor(4.7);  //

Math.sin() и Math.cos()

Math.sin(x) - возвращает синус (значение от -1 до 1) угла x (в радианах).
Math.cos(x) - возвращает косинус (значение между -1 и 1) угла x (в радианах).
Можно использовать градусы вместо радиан, для этого нужно конвертировать градусы в радианы: Угол в радианах = Угол в градусах х PI / 180.
ПРИМЕР 
		
		Math.sin(90 * Math.PI / 180); //  Значение синуса 90 градусов равно 1
		Math.sin(0 * Math.PI / 180);  // 

		Math.cos(90 * Math.PI / 180); // 
		Math.cos(0 * Math.PI / 180);  //

Math.min() и Math.max()

Math.min()и Math.max() - может использоваться, чтобы найти самое низкое или самое высокое значение в списке аргументов:
ПРИМЕР 
		Math.min(0, 180, 43, 55);   
		Math.max(0, 180, 43, 55);

Math.random()

Math.random() - возвращает случайное число от 0 (включительно) до 1 (исключая):
ПРИМЕР 
		Math.random();

Математический конструктор

В отличие от других глобальных объектов, объект Math не имеет конструктора. Методы и свойства являются статическими.
Все методы и свойства (константы) могут использоваться без предварительного создания объекта Math.

random

Math.random()

Math.random() - возвращает случайное число от 0 (включительно) до 1 (исключая), всегда возвращает число ниже 1.
ПРИМЕР 
		Math.random(); //

случайные целые числа

Math.random() используется с Math.floor() может быть использована для получения случайных чисел
ПРИМЕР 
		Math.floor(Math.random() * 10);     // - 0 to 9
		Math.floor(Math.random() * 11);     // - 0 to 10
		Math.floor(Math.random() * 10) + 1; // - 0 to 10

Правильная случайная функция

Эта функция JavaScript всегда возвращает случайное число между min (включенным) и max (исключенным):
ПРИМЕР 
		function random_func_1(min, max) {
		   return Math.floor(Math.random() * (max - min)) + min;
		}
		
		 random_func_1(20, 29) // 

		Эта функция JavaScript всегда возвращает случайное число между min и max (оба включены):
		function random_func_2(min, max) {
		   return Math.floor(Math.random() * (max - min + 1)) + min;
		}
		
		 random_func_2(20, 29) //

Booleans

JavaScript Boolean представляет одно из двух значений: true или false.

Boolean() функция

Boolean() функцию - чтобы узнать, является ли выражение (или переменная) истинным
ПРИМЕР 
		Boolean(10 < 9); // 
		Boolean(10 > 9); // 
		10 > 9;          //

Когда считается true и false

Все со значением считается true. Все без значения считается false.
ПРИМЕР 
		true
		Boolean("Word");   // 
		Boolean(109);      // 
		Boolean(-109);     // 
		Boolean(-2 - 9);   // 

		false
		Boolean(0);        // 
		Boolean(-0);       // 
		Boolean("");       // 
		Boolean();         // 
		Boolean(null);     // 
		Boolean(false);    // 
		Boolean(10 / "H"); //

Логические значения могут быть объектами

Обычно логические значения JS являются примитивными значениями, созданными из литералов: var x = false;
Но логические значения также могут быть определены как объекты с ключевым словом new: var y = new Boolean(false);
Не нужно создавать логические объекты. Это замедляет скорость выполнения, ключевое слово усложняет код и может привести к неожиданным результатам: new
ПРИМЕР 
		var x = false;
		var y = new Boolean(false);
		var z = new Boolean(false);
		typeof x; // 
		typeof y; // 

		При использовании == оператора равные логические значения равны
		x == y;   // 

		При использовании === оператора равные логические значения не равны, поскольку === оператор ожидает равенства как по типу, так и по значению.
		x === y;   // 

		Объекты нельзя сравнивать
		z == y;    // 
		z === y;   //

Сравнение и логические операторы

Сравнение и Логические операторы используются для проверки trueили false.

Операторы сравнения

Операторы сравнения используются в логических выражениях для определения равенства или разницы между переменными или значениями.
ПРИМЕР 
		var x = 5;
		== - равно
		x == 7   -  
		x == 5   -  
		x == "5" -  

		=== - равное значение и равный тип
		x === "5" -  
		x === 5   -  

		!= - не равно
		x != "5" -  
		x != 5   -  
		x != 8   -  

		!== - не равное значение или не равный тип
		x !== "5" -  
		x !== 5   -  
		x !== 8   -  

		> - больше чем (x больше /число/)
		x > 5 -  
		x > 8 -  
		x > 4 -  

		< - меньше чем (x меньше /число/)
		x < 5 -  
		x < 8 -  
		x < 4 -  

		>= - больше или равно
		x >= 5 -  
		x >= 8 -  
		x >= 4 -  

		<= - меньше или равно
		x <= 5 -  
		x <= 8 -  
		x <= 4 -

Как это можно использовать

Операторы сравнения могут использоваться в условных выражениях для сравнения значений и выполнения действий в зависимости от результата
ПРИМЕР 
		if (age < 18) text = "Too young";

Логические Операторы

Логические операторы используются для определения логики между переменными или значениями.
ПРИМЕР 
		var y = 3;
		&& - И
		(x > 3 && y < 5); //  
		(x < 2 && y > 8); //  
		(x < 6 && y < 4); //  

		|| - ИЛИ
		(x > 3 || y < 5); //  
		(x < 2 || y > 8); //  
		(x < 6 || y < 4); //  

		! - НЕ
		!(x > 3);  //  
		!(x < 2);  //  
		!(x || y); //

Сравнение разных типов

Сравнение данных разных типов может дать неожиданные результаты. При сравнении строки с числом JS преобразует строку в число при сравнении.
Пустая строка преобразуется в 0. Нечисловая строка преобразуется в NaN всегда false
ПРИМЕР 
		2 < 12;      // 
		2 < '12';    // 
		2 < "John";  // 
		2 > "John";  // 
		2 == "John"; // 
		"2" < "12";  // 

		При сравнении двух строк «2» будет больше «12», потому что (в алфавитном порядке) 1 меньше 2.
		"2" > "12";  // 
		"2" == "12"; // 

		Чтобы обеспечить правильный результат, переменные должны быть преобразованы в правильный тип перед сравнением:

		function function_different_types() {
		var age, voteable;
		age = Number(document.getElementById("age_2").value);
		if (isNaN(age)) {
		   voteable = "Input is not a number";
		} else {
		   voteable = (age < 18) ? "Too young" : "Old enough";
		}
		document.getElementById("different_types_9").innerHTML = voteable;
		}
