Объекты JavaScript

Все значения JavaScript, кроме примитивов, являются объектами.
Объекты, Функции, Массивы, Регулярные выражения, Математика, Date - всегда объекты
Логические значения, Числа, Строки - могут быть объектами, если они определены с помощью new

Примитивы JavaScript

Примитивным значением является значение, которое не имеет свойств или методов.
Примитивный тип данных представляют собой данные, которые имеют элементарное значение.
JS определяет 5 типов примитивных типов данных: string, number, boolean, null, undefined
Примитивные значения неизменяемы (они жестко запрограммированы и поэтому не могут быть изменены).
если x = 3,14, можно изменить значение x. Но нельзя изменить значение 3,14.
ПРИМЕР 
		Value	    Type	    Comment
		"Hello"	    string	   "Hello" is always "Hello"
		3.14	    number	    3.14 is always 3.14
		true	    boolean	    true is always true
		false	    boolean	    false is always false
		null	    null (object)   null is always null
		undefined   undefined       undefined is always undefined

Объекты - это переменные

Переменные JS могут содержать одиночные значения.
Объекты тоже переменные, но могут содержать много значений. Записываются как пары имя: значение.
Объект JavaScript - это набор именованных значений
ПРИМЕР 
		var person = "Doe Stive";
		person; //

		var person_1 = {name: "Stive", age:"22"};
		person_1.name; //

Свойства объекта

Именованные значения в объектах JavaScript называются свойствами.
ПРИМЕР 
		Свойство      Значение
		first_name   John
		last_name    Doe

Методы объекта

Методы - это действия, которые можно выполнять с объектами.
Свойства объекта могут быть как примитивными значениями, так и другими объектами и функциями.
Метод объекта является свойством объекта , содержащим определение функции.
Объекты JavaScript - это контейнеры для именованных значений, называемых свойствами и методами.
ПРИМЕР 
		Свойство        Значение
		first_name     John
		last_name      Doe
		color          Blue
		full_name      function () {return this.firstName + "" + this.lastName;}

Создание объекта JavaScript

С помощью JS можно определять и создавать свои собственные объекты. Есть разные способы создания новых объектов:
- Определите и создайте единый объект, используя литерал объекта.
- Определите и создайте единый объект с ключевым словом new.
- Определите конструктор объекта, а затем создайте объекты сконструированного типа.
В ECMAScript 5 объект также может быть создан с помощью функции Object.create().

Использование литерала объекта

Это самый простой способ создать объект JS. Используя литерал объекта, одновременно определяете и создаете объект в одном операторе.
Литерал объекта - это список пар имя: значение (например, age: 50) в фигурных скобках {}.
ПРИМЕР 
		создается новый объект JS с четырьмя свойствами:
		var person_2 = {first_name:"John", last_name:"Doe", age:3, color:"blue"};
		person_2.first_name + "is" + person_2.age + "ears old"; 

		Пробелы и перенос строки не важны. Определение объекта может занимать несколько строк:
		var person_3 = {
		   first_name:"John", 
		   last_name:"Doe", 
		   age:3, color:"blue"
		};
		person_3.first_name + " is " + person_3.age + " ears old";

Использование ключевого слова new

Для простоты, удобочитаемости и скорости выполнения использовать первый (метод литерала объекта).
ПРИМЕР 
		создается новый объект JavaScript с четырьмя свойствами:
		var person_4 = new Object();
		person_4.first_name = "John";
		person_4.last_name = "Doe"; 
		person_4.age = 3;
		color = "blue";
		person_4.first_name + " is " + person_4.age + " ears old";

Объекты JavaScript изменяемы

Объекты изменчивы: к ним обращаются по ссылке, а не по значению.
ПРИМЕР 
		Если человек является объектом, следующее утверждение не создаст копию человека:
		var person_5 = { first_name:"Dou", last_name:"John", age:30 };

		Объект x не является копией человека. x, и person - один и тот же объект.
		Любые изменения в x также изменят person
		var x = person_5; //
		x.age = 4; //
		x.age; //
		person_5.age; //

Свойства объекта JavaScript

Свойства - самая важная часть любого объекта JavaScript.

Свойства JavaScript

Свойства - это значения, связанные с объектом JS.
Объект JS - это набор неупорядоченных свойств.
Свойства обычно можно изменять, добавлять и удалять, но некоторые из них доступны только для чтения.

Доступ к свойствам JavaScript

Выражение должно соответствовать имени свойства. Синтаксис доступа к свойству объекта:
СИНТАКСИС 
		object_name.property;    // person.age
		или
		object_name["property"]; // person["age"]
		или
		object_name[expression]; // x = "age"; person[x]

ПРИМЕР 
		var person_6 = {
		   first_name:"John", 
		   last_name:"Doe", 
		   age:3
		};
		person_6.first_name + " is " + person_6.age + " years old "; //
		person_6["first_name"] + " is " + person_6["age"] + " years old ";; //

JavaScript for...in цикле

Оператор JavaScript for...in перебирает свойства объекта.
Блок кода внутри for...in цикла будет выполняться один раз для каждого свойства.
СИНТАКСИС 
		for (variable in object){
		   // code to be executed
		}

ПРИМЕР 
		Цикл по свойствам объекта:
		var txt = "";
		var x;
		for (x in person_6) {
		   txt += person_6[x] + " ";
		}
		txt; //

Добавление новых свойств

Можно добавить новые свойства к существующему объекту, просто присвоив ему значение.
Предположим, что объект person уже существует - тогда можно дать ему новые свойства:
ПРИМЕР 
		person_6.nationality = "Eng";
		person_6.nationality; //

Удаление свойств

delete Ключевое слово удаляет свойство из объекта: и само имущество. После удаления свойство нельзя использовать до его повторного добавления.
delete Оператор предназначен для использования на свойства объекта. Это не влияет на переменные или функции.
delete Оператор не должен использоваться на заранее определенных свойств объектов JS. Это может привести к сбою приложения.
ПРИМЕР 
		delete person_6.age;
		person_6.age; //

Атрибуты собственности

У всех свойств есть имя. Кроме того, они также имеют ценность. Значение - это один из атрибутов свойства.
Другие атрибуты: перечисляемый, настраиваемый и доступный для записи.
Эти атрибуты определяют, как можно получить доступ к свойству (доступно ли оно для чтения? Или для записи?)
В JS все атрибуты можно читать, но можно изменить только атрибут значения (и только если свойство доступно для записи).
(В ECMAScript 5 есть методы как для получения, так и для установки всех атрибутов свойств)

Свойства прототипа

Объекты JS наследуют свойства своего прототипа.
delete Ключевое слово не удаляет наследуемые свойства, но при удалении свойства прототипа, он будет влиять на все объекты, унаследованные от прототипа.
