Цикл for

Циклы могут выполнять блок кода несколько раз.

Циклы JS

Циклы удобны, если нужно запускать один и тот же код снова и снова, каждый раз с другим значением. Часто это имеет место при работе с массивами
ПРИМЕР 
		var cars = ["Volvo", "Audi", "BMW"];
		var text = " ";

		Вместо этого:
		text += cars[0] + "< br>";
		text += cars[1] + "< br>";
		text += cars[2] + "< br>";

		можно написать это:
		var i;
		for (i = 0; i < cars.length; i++) {
		   text += cars[i] + "< br>";
		}

Различные виды Loops

JavaScript поддерживает различные виды циклов:
ПРИМЕР 
		for      - перебирает блок кода несколько раз
		for/in   - перебирает свойства объекта
		for/of   - перебирает значения повторяемого объекта
		while    - перебирает блок кода, пока указанное условие выполняется
		do/while - также перебирает блок кода, пока указанное условие выполняется

Цикл for


СИНТАКСИС 
		for (statement 1; statement 2; statement 3) {
		  // code block to be executed
		}
Оператор 1 выполняется (один раз) перед выполнением блока кода.
Оператор 2 определяет условие для выполнения блока кода.
Оператор 3 выполняется (каждый раз) после выполнения блока кода.
ПРИМЕР 
		var text_1 = "";
		for (i = 0; i < 5; i++) {
		   text_1 += "The number is " + i + "< br>";
		}
		(var i = 0) - устанавливает переменную перед началом цикла 
		(i < 5)     - (я должен быть меньше 5) определяет условие для запуска цикла.
		(i ++)      - увеличивает значение  каждый раз, когда выполняется блок кода в цикле.

Оператор 1

Обычно используют оператор 1 для инициализации переменной, используемой в цикле (i = 0).
Заявление 1 не является обязательным. Можно инициировать много значений в операторе 1 (через запятую):
ПРИМЕР 
		var cars = ["Volvo", "Audi", "BMW"];
		var i, len, text_2;
		for (i = 0, len = cars.length, text_2 = ""; i < len; i++) {
		   text_2 += cars[i] + "< br>";
		}
	

		И можно опустить оператор 1 (например, когда значения установлены до запуска цикла)
		var cars = ["Volvo", "Audi", "BMW"];
		var i = 1;
		var len = cars.length;
		var text_3 = "";
		for (; i < len; i++) {
		   text_3 += cars[i] + "< br>";
		}

Оператор 2

Часто утверждение 2 используется для оценки состояния исходной переменной. Это не всегда так, заявление 2 также необязательно.
Если оператор 2 возвращает истину, цикл начнется заново, если он вернет ложь, цикл закончится.
Если пропустить оператор 2, нужно обеспечить разрыв внутри цикла. В противном случае цикл никогда не закончится. Это приведет к краху браузера.

Оператор 3

Часто утверждение 3 увеличивает значение начальной переменной. Утверждение 3 является необязательным, например когда увеличивается значения внутри цикла.
Оператор 3 может делать что-либо, например, отрицательное приращение (i--), положительное приращение (i = i + 15) или что-то еще.
ПРИМЕР 
		var cars = ["Volvo", "Audi", "BMW"];
		var i = 0;
		var text_4 = "";
		var len = cars.length;
		for (; i < len; ) {
		   text_4 += cars[i] + "< br>";
		   i++
		}

For / In Loop

for/in - Оператор JS проходит через свойства объекта:
ПРИМЕР 
		var person = { fname: "John", lname: "Doe", age:25 };
		var text_5 = "";
		var x;
		for (x in person) {
		   text_5 += person[x] + " ";
		}

For / Of Loop

Оператор JS for/of проходит через значения итерируемых объектов.
for/of позволяет циклически повторять структуры данных, такие как массивы, строки, карты, списки узлов и т. д.
СИНТАКСИС 
		for (variable of iterable) {
		  // code block to be executed
		}
переменная - для каждой итерации переменной присваивается значение следующего свойства.
Переменная может быть объявлена с const, let или var. Итерируемый - объект, имеющий итеративные свойства.
ПРИМЕР 
		Цикл над массивом
		var cars = ["Volvo", "Audi", "BMW"];
		var x;
		for (x of cars) {
		   document.write(x + "< br >");
		}
	


		Цикл над строкой
		var string = "JavaScript";
		var x;
		for (x of string) {
		   document.write(x + "< br >");
		}

While цикл

Циклы могут выполнять блок кода, пока указанное условие выполняется.

Цикл While

Цикл while проходит по блоку кода до тех пор, пока указанное условие выполняется.
Если не увеличить переменную, используемую в условии, цикл никогда не закончится. Это приведет к краху браузера.
СИНТАКСИС 
		while (condition) {
		  // code block to be executed
		}

ПРИМЕР 
		код в цикле будет выполняться снова и снова, пока переменная (i) меньше 10:
		var text_while = "";
		var i = 0;
		while (i < 10) {
		   text_while += "The number is " + i + "< br>";
		   i++;
		}

Цикл Do / While

do/while - Цикл представляет собой вариант цикла.
Этот цикл будет выполнять блок кода один раз, прежде чем проверять, выполнено ли условие, а затем повторять цикл, пока условие выполняется.
СИНТАКСИС 
		do {
		  // code block to be executed
		}
		while (condition);

ПРИМЕР 
		Цикл всегда будет выполняться по крайней мере один раз, даже если условие ложно, потому что кодовый блок выполняется перед проверкой условия:
		var i = 0;
		var text_do_while = "";
		do {
		   text_do_while += "The number is " + i + "< br>";
		   i++;
		}
		while (i < 10);

Сравнения For и While

цикл while очень похож на цикл for, но оператор 1 и оператор 3 опущены.
ПРИМЕР 
		var cars = ["Volvo", "Audi", "BMW"];
		var i = 0;
		var text = "";

		for цикл для сбора имен автомобилей из массива cars:
		for (; cars[i];) {
		   text += cars[i] + "< br>";
		   i++;
		};
	


		while цикл для сбора имен автомобилей из массива cars:
		while (cars[i]) {
		   text += cars[i] + "< br>";
		   i++;
		}

JavaScript Break and Continue

break - Заявление «выпрыгивает» из цикла.
continue - Утверждение «перепрыгивает» на одну итерацию в цикле.

Заявление Break

Оператор break также может быть использован для выхода из цикла. Он разрывает петлю и продолжает выполнение кода после цикла (если таковые имеются):
ПРИМЕР 
		var text = ""
		for (i = 0; i < 10; i++) {
		   if (i === 3) { break; }
		   text += "The number is " i + "< br>";
		}

Заявление continue

continue - Заявление ломается одна итерация (в цикле), если происходит определенное условие, и продолжается со следующей итерации цикла.
ПРИМЕР 
		Этот пример пропускает значение 3:
		var text = "";
		var i;
		for (i = 0; i < 10; i++) {
		   if (i === 3) { break; }
		   text += "The number is " i + "< br>";
		}

JavaScript Labels

Чтобы пометить операторы JavaScript, перед оператором нужно указать имя метки и двоеточие.
Операторы break и continue являются единственными операторами JavaScript, которые могут «выпрыгнуть» из блока кода.
continue Утверждение (с или без ссылки этикетки) могут быть использованы только пропустить одну итерацию цикла.
Оператор break без ссылки на метку может быть использован только для выхода из цикла или переключателя.

СИНТАКСИС 
		break labelname;
		continue labelname;

ПРИМЕР 
		label:
		statements

		С помощью ссылки на метку оператор break может использоваться для перехода из любого блока кода :
		var cars = ["Volvo", "Audi", "BMW"];
		var text = "";
		list: {
		   text += cars[0] + "< br>";
		   text += cars[1] + "< br>";
		   break list;
		   text += cars[2] + "< br>";
		   text += cars[3] + "< br>";
		}
